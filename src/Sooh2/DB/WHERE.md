
## WHERE

本文介绍where的基本写法，为了便于理解，使用mysql的where做对照


* 『 & 』 AND 
* 『 | 』 OR  
* 『 = 』 =； in (...) ; is null
* 『 ! 』 <> ; not in (...) ; is not null
* 『 > 』 > 
* 『 ] 』 >= 
* 『 < 』 <  
* 『 [ 』 <= 
* 『 * 』 like   



|asdf | asdf |
|----------------| -------------------|
| [abc=>1]       | where abc = 1
